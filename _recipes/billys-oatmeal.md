---
title: Billy's Oatmeal
author: Billy
ingredients:
  - { qty: "40g", item: "Oats" }
  - { qty: "-", item: "Oat milk" }
  - { qty: "100g", item: "Frozen blueberries" }
  - { qty: "-", item: "Maple syrup" }
equipment:
  - { type: "1", item: "Coffee mug" }
  - { type: "1", item: "Scale" }
ingredientsYouHave:
  - (Optional) Protein powder
---
- Add 40g of oats to a coffee mug
- (Optional) Add protein powder, mix in
- Add oat milk until it rises above the oats
- Microwave for 1 min
- Stir in 100g of frozen blueberries
- Drizzle with maple syrup and serve
